# WP Skeleton Theme

> This is a work in progress. Very alpha, subject to sweeping changes and random intervals :)

## Installation

#### Via composer

```
{
    "require" : {
        "ptahdunbar/wp-skeleton-theme" : "dev-master"
    }
}
```

#### Via git

```
git clone git@github.com:ptahdunbar/wp-skeleton-theme.git
```
